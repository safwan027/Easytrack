
EasyTrack is a productivity-focused Chrome Extension designed to streamline ticket management directly within WhatsApp Web. It allows users to track, manage, and organize customer interactions without leaving the chat interface.

## 🚀 Getting Started

Since this extension is in active development, you can run it locally by loading it as an "Unpacked" extension in Google Chrome.

### Prerequisites
* **Google Chrome** (or any Chromium-based browser like Brave or Edge).
* A code editor (like VS Code) if you wish to modify the code.

### Installation
1. **Download the Project:**
   - Click the green **Code** button at the top of this repository and select **Download ZIP**.
   - Extract the ZIP file to a folder on your computer.
2. **Load in Chrome:**
   - Open Chrome and navigate to: `chrome://extensions/`
   - Enable **Developer mode** using the toggle in the top-right corner.
   - Click the **Load unpacked** button.
   - Select the extracted folder (the one containing the `manifest.json` file).
3. **Launch:**
   - Open [WhatsApp Web](https://web.whatsapp.com). EasyTrack will automatically load alongside your chats.

## 🛠️ Development Workflow

EasyTrack is designed for easy customization. To make changes:

1. **Modify:** Open the project folder in your IDE and edit the files.
2. **Save:** Save your changes.
3. **Reload:** 
   - Go back to `chrome://extensions/`.
   - Find the **EasyTrack** card and click the **Reload (↻)** icon.
4. **Refresh:** Refresh your WhatsApp Web tab to see your changes in action.
